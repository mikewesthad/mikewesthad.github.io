Augmented Interface (2013) Source Code

This was an experiment in augmented interfaces for game spaces. In addition to a keyboard and mouse, players also use two physical LEDs as controls in the game space.

The game is built using the Unity game engine, openFrameworks C++ toolkit and an Ardiuno board with two LEDs.

The video below shows the installation running. For the purposes of sharing, there is also a version of the game that does not require Arduinos and LEDs below, playable online here.
