Introduction to Graphics (now)

openFrameworks is an open source C++ toolkit to make creating things with code easier. ofBook is a community-sourced book covering the expanse of things you can do with openFrameworks: from animation to iOS applications to game design.

I wrote Chapter 4, "Introduction to Graphics." It is a chapter aimed at beginners who have read the preceding chapters, which cover how to create an openFrameworks application and the basics of C++ programming. My chapter walks the reader through the world of 2D graphics by creating complex forms from repeated, simple shapes.

The completed book will be published (in print and to the web) very soon, but for now, the unformatted version of my chapter can be seen here.
