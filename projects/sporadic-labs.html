Overview

Work-in-progress JS game, in collaboration with Rex Twedt. See public repository for the latest details. Built with phaser, babel, webpack, preact and mobx.

Libraries

In building the game, I created a series of libraries for JS game development: Phaser Tiled Hull (library for simplifying shapes) and Phaser Navmesh (library for super fast pathfinding).
