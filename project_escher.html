Escher (2012)

Escher was made in 48-hour for the Ludum Dare game jam, for the theme "fear." It is a short story about an elderly man home alone.

It is a flash game built with the Flixel library. A playable version is hosted online here. The left and right arrow keys control movement, while the space bar controls interactions.
